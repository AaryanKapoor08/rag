# RAG Learning Workspace

This repository contains hands-on notebooks and sample files for learning retrieval-augmented generation workflows with LangChain, embeddings, and vector stores.

## Contents

- `0-DataIngestionParsing/`: text, PDF, Word, CSV, Excel, JSON, and SQLite ingestion examples
- `1-VectorEmbeddingsAndDatabases/`: embedding concepts and OpenAI embedding examples
- `2-VectorStores/`: FAISS, Chroma, Pinecone, Astra DB, and in-memory vector store examples
- `main.py`: lightweight sanity check for the workspace layout

## Run

```powershell
uv run python main.py
```

The command checks that the core learning sections exist and prints a short inventory.
